My Projects

AURA

I built an AI-powered RAN analytics platform using Cloudflare Workers, Durable Objects, and D1. The system continuously logs network KPIs such as latency, throughput, and congestion, storing thousands of records for real-time analysis. By integrating Meta’s Llama 3.3 model, it allows users to query the data conversationally and instantly getting an AI-generated insight.

SCRUB

I developed a reinforcement learning agent using Deep Q-Networks (DQN) in Python to simulate autonomous navigation within a customizable GridWorld environment. The project modeled real-world decision-making in dynamic, driving-like scenarios

Rare disease prediction

I developed a hypergraph-based methodology to prioritize genes based on patient symptoms and predict rare diseases. This work was recognized as the Best Capstone Project at PES University and published in Scientific Reports.

YAK

I built a high-performance messaging system using a Producer–Subscriber architecture to simulate large-scale market data streaming and real-time event processing. The system is designed for low latency and fault tolerance, ensuring reliable performance under heavy loads.
